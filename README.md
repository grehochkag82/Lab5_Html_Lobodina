### Тэги в HTML
Структура парного тэга: 
**<тег>тело</тег>**

*Пример:*
```
<h1>это заголовок</h1>
```
<h1>Лабоаторная работа №5</h1>
    <p>Это моя первая HTML-страница, созданная в рамках изучения основ веб-разработки</p>
    
-------
______
<h2>Это подзаголовок</h2>
<h3>Подраздел</h3>
<p>Это обычный текст абзаца</p>
<p>Это<strong> очень важная</strong> фраза</p>
<p>Это<em> курсивный</em> текст</p>
<p>Линия 1 <br>Линия 2</p>
<hr>
<p><a href = "https://google.com">Перейти на Google</a></p>

<p><img scr="ex.png" alt="Картинка не загрузилась"/></p>
